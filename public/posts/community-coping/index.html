Inspiration

This was a group project completed as part of an Interactive Product Design Studio. The focus area was Tackling COVID-19 Misinformation through Design. Rather than tackling misinformation directly we reframed the brief:

How could we improve the capacity of communities to better cope with the impact of such information

Ideation

  1. Ideated several concepts
  2. Collected user feedback
  3. Iterated and refined to a single concept
  4. Usability testing
  5. High-fidelity prototype

Implementation

The web application was developed as a mobile-first static website with front-end code. It was developed in this way more as a proof of concept with the opportunity to develop the backend at a later stage.
